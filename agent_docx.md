# 2026“智创未来”华中科技大学AI智能体大赛项目说明书

**项目名称：** MacMate —— 基于自然语言驱动的MacOS桌面智能助理 

**参赛命题：** 命题F：自选命题（智能办公与综合领域）

## 一、 项目背景与立项宗旨

在日常的数字化工作流中，MacOS用户面临着应用碎片化、系统资源管理繁琐、日程与实际行为脱节等痛点。传统的效率工具往往需要用户主动去适应复杂的UI和设定规则。 本项目旨在开发一款名为“MacMate”的桌面级AI智能体。它不仅是一个“听从指令”的工具，更是一个具备“感知、推理、执行”能力的“主动型”助理。MacMate通过深度集成MacOS底层API和命令行工具，结合大语言模型的逻辑推理能力，旨在实现全场景的自然语言系统控制、智能系统监控与个性化数字生活管理，真正让AI技术赋能大学生的科研、开发与日常学习。

## 二、 核心功能模块详解（Agent Skills）

为了实现全方位的接管，MacMate 的功能被划分为四大核心模块：

### 1. 系统感知与状态监控模块 (System & Health)

Agent 具备对所处宿主机（MacOS）环境的全面感知能力，这是进行推理和执行动作的基础。

- **电脑状态实时监控：** 实时读取并分析系统的硬件资源占用（如 Disk Usage、RAM 占用、CPU 温度及负载等）。
  - *主动干预：* 当检测到内存告急或某进程（如未知的后台进程）占用异常飙升时，Agent 会主动提示用户检查activity monitor。
- **用户活动静默监控：** 在获取用户授权的前提下，记录前台活跃窗口的切换频率、特定应用（如 IDE、浏览器、文档编辑器）的使用时长。
  - *数据脱敏：* 所有活动数据仅在本地进行加密向量化存储和处理，确保个人隐私安全。

### 2. 个人效能与日程管理模块 (Productivity & Schedule)

将用户的碎片化时间与任务目标进行对齐，提供智能化的时间管理。

- **自然语言日程安排：** 用户输入今日任务，agent根据其推理能力和用户使用习惯自动进行计划，生成日程。
- **基于活动的“今日简报”生成：** 每日固定时间（如晚上 10 点），Agent 根据“用户活动监控”模块收集的数据，结合 LLM 的总结能力，生成一份结构化的 Markdown 简报。
  - *简报内容：* 包含今日核心工作时长（如“您今天在代码编辑器上投入了 4 小时”）、摸鱼时间占比分析、以及对明天日程的初步梳理与建议。

### 3. 场景自动化与预设指令模块 (Automation & Workflows)

允许用户通过极简的指令，触发涉及多个应用的复杂系统级联动。

- **场景化预设指令 (Profiles)：**
  - *“专注开发模式”：* 自动开启 MacOS 勿扰模式，关闭社交软件（微信、QQ），打开终端应用并清屏，启动常用的代码编辑器和本地测试服务器。
  - *“下班休闲模式”：* 一键关闭所有办公软件，清空剪贴板，打开视频客户端或游戏。
- **智能终端与系统设置助手：** 允许用户用自然语言完成复杂的命令行操作或系统配置。例如输入“帮我把 WiFi 断开重连一下”或“找出占用 8080 端口的进程并杀掉”，Agent 会自动将其转换为对应的 Shell 命令并在后台执行。

### 4. 情绪价值与娱乐生活模块 (Lifestyle)

赋予 Agent 人文关怀，使其不仅是冰冷的效率工具。

- **上下文感知的智能选歌：**
  - Agent 能够根据当前的系统时间、天气状态（调用天气 API），甚至是用户当前的活动状态（如正在写代码还是在深夜浏览网页），推断用户的心境。
  - 结合推理结果，自动调用 Apple Music 或 Spotify 的本地控制接口（通过 AppleScript），播放如“深夜 Lo-Fi 编码歌单”或“清晨活力流行乐”，提供沉浸式的伴随体验。

## 三、 系统架构与技术实现路线

1. **交互层 (Frontend)：** 采用轻量级的状态栏菜单 (Menu Bar App) 或悬浮搜索框设计，支持快捷键唤醒，实现类似Siri的体验。
2. **推理层 (Brain - LLM)：** 接入模型 API或本地ollama模型，通过精心设计的 System Prompt 和 Function Calling (工具调用) 能力，让大模型判断用户的自然语言需要映射到哪个具体的功能函数。
3. **执行层 (Action - OS API)：** 这是 MacMate 的核心技术壁垒。
   - 利用 Python 强大的生态（如 `psutil` 用于系统监控）。
   - 通过 `os.system` 或 `subprocess` 模块执行底层 Shell 命令。
   - 通过封装的 `osascript` (AppleScript) 与 MacOS 独有的图形化应用（如日历、音乐、系统设置）进行深度交互。

## 四、 项目创新性与参赛优势

1. **突破“沙盒”限制：** 目前市面上多数 AI 智能体停留在网页端或对话框内。MacMate 直接扎根于操作系统底层，具备真实的“行动力”（Action），完美契合大赛“感知-推理-执行”的命题要求。
2. **极强的实用与推广价值：** 无论是自动化文件管理、智能终端助手，还是日常的活动复盘，都直击学生群体和极客开发者的真实痛点，属于高频、刚需应用场景。
3. **技术栈落地可行性高：** 结合 Python 处理逻辑与数据，以及 MacOS 原生的系统控制脚本，能够在比赛周期内快速迭代出具有高完成度的 MVP (最小可行性产品)。
